## Key Competitive Insights

### What Miro does best
- 16-color sticky palette (vs our 6)
- Live cursor tracking with user labels
- Extensive template marketplace
- Auto-layout and mind map tools

### What Milanote does best
- Beautiful card design with content previews
- Rich media mixing (images, text, links, files)
- Flexible grid with free-form placement

### What ThreadDeck does best
- Thread-based navigation paradigm
- Visual hierarchy indicators
- Nested content with expandable threads

### Our competitive advantages
- Specialized node types (Map, TripPlanner, CodeBlock)
- Firebase cloud sync infrastructure
- Global theming system (in progress)
- Integrated template system (6 templates)

## Scoring Methodology

Each dimension scored 1-5:
- **1:** Missing or non-functional
- **2:** Basic implementation, below expectations
- **3:** Functional, meets minimum expectations
- **4:** Good implementation, above average
- **5:** Best-in-class, reference implementation

Scores based on public product analysis, documentation review, and feature comparison against current UnityNOTES production state.
