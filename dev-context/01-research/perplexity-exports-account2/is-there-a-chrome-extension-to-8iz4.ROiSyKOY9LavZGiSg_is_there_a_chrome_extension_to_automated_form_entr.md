# Is there a chrome extension to automated form entries with dummy data for testing purposes

Yes, there are several Chrome extensions available for automating form entries with dummy data for testing purposes. Here are some notable options:

## Fake Data

Fake Data is a highly advanced Chrome extension for filling forms with random and fake data[^7]. It offers the following features:

- Generates random names, emails, addresses, phone numbers, and more
- Allows creation of custom data types using JavaScript
- Fills single fields or entire forms at once
- Uses keyboard shortcuts for faster operation
- Includes an auto-learn feature for repeated form filling


## Fake Filler

Fake Filler is another popular extension designed to streamline form testing[^6]. It automatically fills all input fields on a page with dummy data and offers adjustable settings to suit different testing requirements.

## QuickForm

QuickForm is a powerful autofill Chrome extension that provides[^8]:

- Adaptive Autofill Technology to fill various form elements
- Auto-clicking feature for smoother form navigation
- Customizable delay options between actions
- Import/export functionality for profiles


## Other Options

1. Form Filler: This extension allows you to fill all form inputs with dummy data[^1].
2. iMacros for Chrome: While not specifically designed for form filling, it can be used to automate repetitive tasks, including form submissions[^4].

These extensions can significantly improve efficiency for developers, QA engineers, and anyone who frequently works with online forms for testing purposes[^3][^5].
